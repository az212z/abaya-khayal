# عباية خيال — موقع تعريفي

موقع تسويقي فاخر (RTL عربي) لمحل **عباية خيال** في تبوك.

## نبذة
- **الاسم:** عباية خيال
- **التصنيف:** محل عبايات فاخرة / تفصيل حسب المقاس
- **المدينة:** تبوك — العزيزية القديمة
- **التقييم:** 4.8 على خرائط قوقل (168 مراجعة)

## سبب الاختيار
نشاط بلا موقع إلكتروني، بتقييم عالٍ (4.8) وحضور بصري قوي (واجهة وهوية وتطريز مميّز)، وخدمات واضحة قابلة للعرض: عبايات جاهزة، تفصيل، شيلات وطرح، عبايات مناسبات، وتطريز — مادة مثالية لموقع راقٍ يدفع للتواصل عبر واتساب.

## فكرة التصميم
هوية "بوتيك عبايات فاخر" تعكس اسم *خيال*: أسود قريب من الفحمي (#1A1719) + ذهبي شامبين (#C6A668) + لمسة موف ناعم (#B49AA6) + عاجي دافئ (#F4EFE9). خطوط عرض أنيقة (El Messiri / Reem Kufi) مع متن نظيف (Tajawal)، إحساس تحريري/أزياء، حركة خفيفة مصنوعة يدويًا (sticky header، scroll-reveal، image zoom، lightbox خفيف).

## الأقسام
هيدر/تنقل · هيرو · شريط ثقة · أصناف العبايات · لماذا خيال · المعرض (صور حقيقية) · نموذج الطلب · الموقع وساعات العمل · دعوة ختامية · فوتر · أزرار عائمة (واتساب/اتصال/خريطة).

## التقنيات
HTML + CSS + JavaScript **vanilla** فقط — بدون أطر أو مكتبات ثقيلة، بدون خطوة بناء. جاهز للنشر على GitHub Pages كما هو.

## النصوص (Copy)
كل النصوص محايدة الجنس (بدون صيغ أمر مؤنثة): "اطلب الآن"، "تواصل معنا"، "اطلب عبايتك".

## حالة الـ Backend
**غير مربوط بخادم.** نموذج الطلب يجهّز رسالة واتساب جاهزة عبر `wa.me/966531900776` ويحفظ نسخة محليًا في `localStorage` (تجريبي) مع إشعار نجاح (toast).

## مصدر الصور
صور حقيقية للنشاط من خرائط قوقل (ملف المشروع). المستخدمة فعليًا (واجهة/تطريز/داخل المحل/عباية):
`ak-261.jpg`، `ak-14790.jpg`، `ak-2030.jpg`، `ak-14477.jpg`.
استُبعدت صور لا تخص النشاط (واجهات محلات أخرى).

## التشغيل محليًا
```bash
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

## رابط الموقع
سيُضاف بعد النشر على GitHub Pages.

## نتائج الاختبار
تُضاف بعد المعاينة الحية (preloader يختفي، القائمة الجوال تعمل كطبقة كاملة، scroll-reveal يظهر، النموذج يفتح واتساب).
